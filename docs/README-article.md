# Статья про сервис Koleyka для WordPress

## Файлы

- **article-koleyka.md** — текст статьи в Markdown со структурой H1/H2 и SEO-блоком
- **article-koleyka-wordpress.html** — готовая вставка для WordPress (HTML между WORDPRESS_START и WORDPRESS_END)

## Картинки для статьи

Сгенерированы три изображения:

1. **koleyka-hero.png** — главный экран: логотип Koleyka, «Очередь на въезд»
2. **koleyka-roles.png** — схема ролей: администратор, руководитель, дежурный, контролёр, водитель
3. **koleyka-queue.png** — экран таблицы очереди с подсветкой строк

Они сохранены в папке assets проекта. Для WordPress загрузите их в Медиафайлы и в HTML статьи подставьте реальные URL вместо `https://ваш-сайт.ru/wp-content/uploads/...`.

## SEO (Yoast / Rank Math)

- **SEO Title:** Koleyka — сервис управления очередью автомобилей на въезд | Учёт и контроль КПП
- **Meta Description:** Koleyka: веб-сервис для учёта очереди машин на въезд. Разные роли: администратор, дежурный КПП, водитель. Статусы, место разгрузки, примечания. Работает в браузере и на смартфоне.
- **Focus Keyphrase:** сервис управления очередью автомобилей

## Как вставить в WordPress

1. Посты → Добавить новый.
2. Переключитесь в режим «Код» / HTML.
3. Скопируйте блок между комментариями WORDPRESS_START и WORDPRESS_END из article-koleyka-wordpress.html и вставьте в редактор.
4. Загрузите картинки в Медиафайлы и подставьте их URL в теги img.
5. В плагине SEO укажите Title, Description и ключевую фразу.
6. Опубликуйте запись.
